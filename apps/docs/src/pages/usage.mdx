# Usage

Learn the basics of working with this and many similar libraries.

## Quickstart react

After [installation](./get-started.mdx#installation), you can import any package compnoent and start using it.

The component configuration is as close to native components as possible, so if you know the basics of react, you should know how to use the components too.
