# Input

The input allow users to enter text into a UI. They typically appear in forms and dialogs.

## Basic Input

The `Input` is a form control integrated with a label with the ability to react to disabling or error.

<Preview component={<Input label="Example" />}>
    ```js
    import { Input } from '@hedgecode/foundation'

    export default function InputWrapper() {
        return <Input label="Example" />
    }
    ```
</Preview>

<Callout type="info">An filled variant is planned to be added in the near future.</Callout>

## Types

The component supports all text fields types includes textarea.

Some types are not supported in this component due to their unique character.
